# TripGenie — Travel Recommendation System & Itinerary Planner

TripGenie is a full-stack web application that generates personalized travel itineraries based on user preferences. It helps users discover places and plan trips efficiently through a structured recommendation system.

## 🚀 Features

- Personalized travel recommendations based on user inputs
- Dynamic itinerary generation using multiple preference parameters
- RESTful API integration between frontend and backend
- Responsive UI for seamless experience across devices

## 🧠 System Design

The application processes user inputs such as budget, interests, and travel style, and converts them into structured decision logic to generate consistent recommendations.

- Rule-based recommendation engine (8+ parameters)
- Modular API design for scalability
- Separation of frontend and backend services

## 🛠 Tech Stack

**Frontend:** React, Tailwind CSS  
**Backend:** Flask (Python)  
**Database:** Supabase  
**Other Tools:** REST APIs, Git
